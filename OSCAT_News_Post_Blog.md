# Meilenstein der Automatisierungsgeschichte gesichert: Das offizielle OSCAT-Legacy-Archiv

Die **Open Source Community for Automation Technology (OSCAT)** hat die Entwicklung von SPS-Software grundlegend geprägt. Durch das Bereitstellen von 100% quelloffenem, hardwareunabhängigem Structured Text (ST) Code wurde das einfache Übertragen von Applikationen zwischen Plattformen wie CODESYS, STEP 7 und PC Worx erst richtig komfortabel. 

Um diesen wertvollen Schatz an historischer Entwicklungsarbeit zu sichern, wurde nun das offizielle Archiv unter der Flagge der Eclipse Foundation bereitgestellt: [eclipse-oscat/oscat-libs-archive](https://github.com/eclipse-oscat/oscat-libs-archive).

## Ein Blick in das Archiv
Das Archiv deckt die drei tragenden Säulen der klassischen OSCAT-Bibliotheken ab:

1. **OSCAT BASIC:** Enthält über Hunderte von Modulen für mathematische Berechnungen, String-Operationen und grundlegende Regelungstechnik.
2. **OSCAT BUILDING:** Das Fundament für HLK- und Gebäudeautomationsprojekte.
3. **OSCAT NETWORK:** Ermöglicht Netzwerkkommunikation direkt aus der SPS heraus.

Mit 55 Releases allein in der BASIC-Bibliothek (von Version 1.10 bis 3.36) dokumentiert das Repository die Evolution der Bibliothek über mehr als ein Jahrzehnt.

## Wichtig für Legacy-Systeme
In der Industrie laufen Anlagen oft jahrzehntelang. Wenn Sie ein bestehendes System warten oder erweitern müssen, sind moderne Bibliotheken manchmal nicht mehr kompatibel. Dieses Archiv stellt sicher, dass Sie:
* Die passenden Dateiformate wie `.lib` (CODESYS V2.3 / TwinCAT V2), `.library` (CODESYS V3) oder S7-Quellarchive finden.
* Zugriff auf historische Revisionshistorien und die damaligen PDF-Dokumentationen (sowohl auf Deutsch als auch Englisch) haben.

## Lizenzrechtliche Besonderheit
Das Archiv ist als schreibgeschützt (*archived*) markiert und wird nicht mehr aktiv weiterentwickelt. Zu beachten ist, dass diese Legacy-Dateien **nicht** unter der neuen Eclipse Public License 2.0 (EPL-2.0) stehen, sondern weiterhin ihren ursprünglichen Lizenzen unterliegen. Diese erlauben jedoch die kostenfreie Nutzung im privaten wie auch im kommerziellen Umfeld.

Besuchen Sie das Repository auf GitHub unter [eclipse-oscat/oscat-libs-archive](https://github.com/eclipse-oscat/oscat-libs-archive) für den direkten Zugriff auf alle Legacy-Downloads.
